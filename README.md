## 🚀 Features

### Core Engine
- **6-Stage NLP Pipeline**: spaCy Transformer NER → Zero-shot Classification (BART-MNLI) → Sentence Embeddings (MiniLM) → FAISS Vector Normalization → ML Proficiency Inference → Ebbinghaus Skill Decay
- **Multi-Agent AI System**: Parser → Analyst → Tutor → Critic agents with self-correcting pipeline
- **Knowledge Graph Engine**: NetworkX DAG with HITS, PageRank, Betweenness Centrality, Community Detection, Critical Path Method
- **Adaptive Pathing Algorithm**: Topological sort with priority scheduling, Coffman-Graham parallel tracks, constraint satisfaction
- **Zero Hallucination**: 500+ curated real courses from verified free providers. LLM never generates course names.

### Intelligence Layer
- **Bayesian Knowledge Tracing (BKT)**: Updates mastery probability after each quiz answer
- **Item Response Theory (2PL IRT)**: Computerized Adaptive Testing with Newton-Raphson ability estimation
- **Bloom's Taxonomy Progression**: Courses mapped to cognitive levels (Remember → Create)
- **Spaced Repetition (SM-2)**: Long-term retention scheduling
- **Skill Decay Modeling**: Ebbinghaus forgetting curve applied to unused skills

### User Experience
- **5 Pathway Views**: Interactive 2D DAG (React Flow), Timeline/Gantt, Kanban, List, 3D
- **Skill Radar Chart**: Current vs. required skills overlay
- **Real-time Streaming**: 14-stage pipeline with live progress visualization
- **Full Reasoning Traces**: Per-recommendation evidence chains
- **AI Chatbot**: Context-aware assistant
- **Gamification**: XP system, achievement badges, learning streaks

### Cross-Domain
Works for 6+ domains: Technology, Data, Business, Operations, Healthcare, Finance
